## Project Overview
This project focuses on analyzing employee task and productivity data to generate
meaningful insights for organizations. The system helps in identifying productivity
trends, performance bottlenecks, and predicting employee performance levels using
basic data science and machine learning techniques.

---

## Problem Statement
Organizations often lack data-driven insights into:
- Employee productivity
- Task completion trends
- Performance bottlenecks

This project aims to address these challenges using analytics and machine learning.

---

## Objectives
- Analyze employee task and productivity data
- Generate performance metrics using data analytics
- Identify productivity trends and bottlenecks
- Predict employee performance levels (High / Medium / Low)
- Visualize insights for decision support
